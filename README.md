# Parth Plastopack Pvt. Ltd. - B2B E-commerce Catalog Hub

A high-conversion, ultra-clean, and responsive B2B E-commerce Catalog website built for **Parth Plastopack Pvt. Ltd.** (Manufacturers of Pharma & Nutraceutical Plastic Packaging). Engineered for lead capture, technical specification delivery, and instant RFQ (Request for Quote) submissions.

## 🚀 Key Features

* **Branded Animated Preloader:** Custom branding screen animating load progression from 0% to 100% on reload.
* **3D Perspective Hover Cards:** Products render inside interactive cards that tilt and project in 3D space on hover.
* **Integrated Lead Generation Drawer:** Sidebar RFQ drawer contextually populated with selected product dimensions and material configurations.
* **Dynamic Category Filters:** Fast tag-based navigation containing real product icons and category visualizers.
* **Technical Spec Sheets:** Clean modal overlays detailing polymer resins (HDPE, PET, PP, LDPE), ISO 15378 compliance metrics, and active US-FDA Drug Master File registry numbers.
* **B2B Call-to-Actions:** Integrated Direct WhatsApp chat button and instant email forms.

## 🛠️ Technology Stack

* **Core Framework:** React 18 (Vite Bundler)
* **Design Systems & Components:** Ant Design (antd)
* **Styling Layer:** Tailwind CSS
* **Icons:** Lucide Icons / Ant Design Icons
* **Static Fallback:** Single-page CDN version (`public_html/catalog.html`) for offline/standalone execution.

## 💻 Local Installation & Setup

Ensure you have Node.js (LTS version) installed on your system.

1. **Install Dependencies:**
   ```bash
   pnpm install
   ```
   *(or `npm install` / `yarn install` depending on your package manager)*

2. **Run Development Server:**
   ```bash
   pnpm run dev
   ```

3. **Build for Production:**
   ```bash
   pnpm run build
   ```
